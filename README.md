# pedalboys-simulink
본 내용은 Topping e2xz2 otg 모델을 사용하여 개발되었습니다.
사용하시는 오디오 인터페이스의 드라이버 모델과 맞춰서 설정을 바꾸어주세요.

초기 인풋/ 아웃풋 설정은 다음과 같습니다.
input : 
Driver : ASIO
Sample rate : 44100
채널 개수 : 1
프레임당 샘플 수 : 128

*프레임당 샘플 수가 과도히 높아지거나 낮아지면 레이턴시가 발생할 수 있습니다.

Output : 
Driver : DirectSound
장치 : Playback 3/4(E2x2 OTG)
* (입력에서 샘플 레이드 상속) 칸에 체크 요망


본 프로젝트는 실제 아날로그 기타 이펙터(TS808, DS-1, RAT)의 회로 구조를 기반으로, 해당 회로를 Simulink 블록과 MATLAB Function 코드로 디지털화하여 구현한 교육용 DSP 플랫폼이다.
기존의 아날로그 회로는 구성요소(R, C, 다이오드)의 역할을 시각적으로 이해하기 어렵고, 각 소자가 소리 변화에 미치는 영향을 직관적으로 체감하기 쉽지 않다. 본 모델은 이러한 한계를 해결하기 위해 설계되었으며, 회로 → DSP 블록 → 실제 코드 → 출력 파형/사운드라는 단일한 흐름으로 학습 및 실험이 가능하도록 구성되어 있다.

1. 실제 회로 기반의 디지털 이펙터 모델링

TS808, BOSS DS-1, ProCo RAT 세 가지 대표적인 드라이브 이펙터의 회로를 분석하여, 각 회로의 동작을 Simulink에서 1:1로 재현하였다.
Soft clipping, Hard clipping, 하이패스·로우패스 필터, Slew-rate 제한 등 각 페달 특유의 회로적 특성을 그대로 DSP 알고리즘으로 변환하여 구현하였다.

2. 소리의 변화가 즉시 나타나는 실시간 시뮬레이션

Drive, Tone, Level 파라미터를 조정하면 그 값이 즉시 DSP 파이프라인에 반영되며,
결과 파형과 사운드가 즉각적으로 변한다.
사용자는 파라미터 조합과 코드의 변화가 소리에 어떤 영향을 주는지 즉시 확인할 수 있다.

3. Soft/Hard Clipping 알고리즘 완전 재현

TS808: OPAMP 피드백 루프의 대칭 다이오드 구조를 기반으로 한 Soft Clipping
RAT: Silicon diode 기반의 Hard Clipping
DS-1: Soft → Hard Clipping의 이중 단계 구조
이 세 가지 클리핑 방식을 DSP 코드로 각각 고유하게 구현하여, 실제 페달 특유의 질감과 톤을 충실히 재현했다.

4. Tone·Volume회로의 DSP 기반 구현

Tone 파트는 고역·저역 통과 필터의 가중치를 가변저항처럼 조절하는 방식으로 구현했다.
Volume은 출력 Gain으로 단순화하여 실제 회로의 구조와 동일한 방식으로 동작하게 한다.
이로 인해 학생들은 저항값 변화 → 필터 응답 변화 → 사운드 변화의 흐름을 직관적으로 이해할 수 있다.

5. 높은 확장성: 새로운 페달을 자유롭게 추가 가능

본 모델의 핵심은 하나의 페달을 만드는 것이 아니라,
“어떠한 페달도 만들 수 있는 DSP 구조를 만드는 것”에 있다.
소리 shaping은 모두 코드 기반이기 때문에 Soft/Hard/Saturation/Filter 계열의 알고리즘을 조합하여
원하는 이펙터를 자유롭게 확장 가능하다.
새로운 상용 페달 복각 또한 가능하며, 코드 몇 줄 또는 파라미터 조정만으로 전혀 새로운 사운드를 만들 수 있다.

Educational Purpose

본 프로젝트는 음악 + 코딩 + 회로전자를 하나의 학습 경험으로 연결한다.
특히 공대 초심자에게 어려운 RC 필터·다이오드·증폭 회로 등의 개념을
실제 소리 변화를 통해 직관적으로 이해할 수 있도록 설계했다.
Simulink의 모델 구조를 보며 회로적 의미를 배우고, MATLAB Function 내부 코드를 수정해 DSP 동작을 직접 실험하며,
그 결과를 파형·사운드로 즉시 확인하는 구조의 학습 환경을 제공한다.
